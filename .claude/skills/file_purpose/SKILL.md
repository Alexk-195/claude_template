## Creating new folders

If you create a new folder add also a file named `.folder_purpose.md` to the folder. 
In the first line add a meaningfull title what the folder is for, start the line with "#".
Optionally add more description below

## Creating new files

If you create a source code file or a script and the language supports comments, add a comment line with `@file_purpose` tag following file purpose. Put this line as first or second line if possible or somewhere in first 20 lines.

If you create a md documentation file, put a meaningfull title in first line starting with"#"
